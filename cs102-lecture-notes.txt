git clone  -  clones the repository into the local machine
git remote -v  -  check whether the remote origin is already be setup by running
git add .  -  stages the files to the repository
git commit -m "type a message to name the changes"  -  commit helps to store changes in the git repository permanently.
git status  -  tells the current status of the repository
